# ESLint Config

The eslint config reused in JValue projects.

## How to use

Install via `npm install --save-dev @jvalue/eslint-config-jvalue`.

**.eslintrc** file:
```javascript
{
    "extends": "@jvalue/eslint-config-jvalue"
}
```


**.prettierrc** file:

```javascript
{
  "singleQuote": true,
  "trailingComma": "all"
}
```
